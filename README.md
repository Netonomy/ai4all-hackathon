# Netonomy: Become your own Server

Netonomy aims to help people and organizations take control of their digital identity, data and finances.

## 🤯 Features

- 💻 Your own Server and API
- 💽 Decentralized Web Node
- ₿ Bitcoin wallet
- ⚡️ A lightning node
- 🤖 Autonomous Agent
